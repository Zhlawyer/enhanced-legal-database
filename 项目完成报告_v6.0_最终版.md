## 🎯 核心功能演示

### 1. 法条拆分功能
```python
# 输入法律文本，自动拆分为独立条款
splitter = ArticleSplitter()
articles = splitter.split_detailed(legal_text)
# 输出: 条、款、项、目层级结构
```

### 2. 语义搜索功能
```python
# 基于向量相似度的智能搜索

```
